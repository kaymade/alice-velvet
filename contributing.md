# Contributing to Alice Velvet

Thank you for your interest in improving Alice Velvet.

## Reporting Issues

If you find a bug, please open a GitHub issue and include:

- Your Obsidian version
- Your operating system
- Whether you are using dark or light mode
- A screenshot if possible
- Steps to reproduce the issue

## Suggesting Changes

Suggestions are welcome. Please describe what you want changed and why it would improve the theme.

## Pull Requests

Before opening a pull request, please make sure:

- The theme still works in both dark and light mode
- `theme.css` does not use `!important`
- `manifest.json` remains valid JSON
- Version numbers follow `x.y.z`
- Changes preserve the overall Alice Velvet aesthetic

## Contact

For questions or feedback, contact `sagacloud` on Discord.
